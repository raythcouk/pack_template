# Addon Pack Template
Contains the following files:

template_b (folder)
- pack_icon.png
- manifest.json
- texts (folder)
  - languages.json
  - en_US.lang

template_r (folder)
- pack_icon.png
- manifest.json
- texts (folder)
  - languages.json
  - en_US.lang

This is designed to provide the minimum files for creating an addon. The manifest simply needs UUID's added, and then you can edit pack name in the en_US.lang files
